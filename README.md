# Pure TypeScript Cli

## about

forked from [Starting a TypeScript Project in 2021](https://www.metachris.com/2021/04/starting-a-typescript-project-in-2021/)

## usage

- cli 
```shell
yarn cli
```

- test
```shell
yarn test
```

- lint
```shell
yarn lint
```

- build
```shell 
yarn build
```

- starting as `ts-node` project
```shell
yarn ts-node <filename>
```
